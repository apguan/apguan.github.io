Pride Softwares

Pride is a hackathon project built at HackDavis 2016 to help Teach for America tackle the challenge teachers face when creating lesson plans. It uses IBM's Watson to help parse through data and allows teachers to give feedback to one another. The build wasn't all buttery smooth though! We had some challenges along the way. Originally, we wanted to create the project using a MEAN stack because of all the CRUD operations (no one on the team had done this before). Eventually, we decided we would figure it out some other time, and build the front-end with static HTML and the back-end with Python. In the end won not only Best Education Tool, but also Best Use of Cisco Spark at HackDavis.

SpeakEz

SpeakEz is a project inspired by the fear of public speaking. Using an HTC Vive and Leapmotion sensor, my team developed a virtual reality simulator to help people practice public speaking in front of crowds. The simulator recognizes tone inflections and hand gestures, all traits that attribute to what makes a speaking performance.

Electric Longboard

So this is more of a personal project that I'm excited to share. Ever since I was a kid, I've wanted a motorized longboard. Do you remember the lawnmower engine-powered skateboards from the 90s? If you do, you'll remember how impractical and terribly loud they were. Now, with high-powered brushless motors, it's easier than ever to mount an electric motor onto a longboard. The controller for the board is a Wii-mote, which has been hacked to connect to an Arduino, which acts as a bluetooth receiver for the electronic speed controller (ESC). The benefits were obvious over a stock controller. It meant a fully customized ESC that can sense changes in terrain and adjusts for power automatically, and one top of that, something sleek and pocketable. The instructions were written in Python, a first for me, but most definitely not the last time (working on getting better). Other than coding, I had to figure out how to wire everything together, and build the deck using composites.
